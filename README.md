# my-configs
".vimrc" and ".tmux.conf" file that configures my development enviroment 

## Installation
    ```
    git clone https://github.com/Arinmis/vim-settings.git
    cd vim-settings
    ./install.sh
    ```
    
## Update repo
To include new settings to repo, run this:   
    ```
    ./update-configs.sh
    ```
